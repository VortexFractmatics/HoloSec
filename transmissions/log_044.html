SSD vs HDD

HDD (Spinning Disk): Easy to wipe. When you write zeros, they stay zeros.

SSD (Flash): Hard to wipe. Due to "Wear Leveling," the drive controller moves data around. Overwriting a file might leave the original copy in a "retired" block that forensic tools can access.

Conclusion: For absolute top-secret data, use an encrypted HDD or physically destroy the SSD after use. HoloSec accounts for this by encrypting the entire container so no unencrypted blocks ever exist.
